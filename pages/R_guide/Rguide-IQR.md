The command `summary(variable)` can be used to get the minimum, 1st IQR, 
median, mean, 3rd IQR, and maximum values.
```R
>summary(spidersconsumed)
	Min.	1st Qu.	Median	Mean	3rd Qu.	Max.
	0	0	0	3	0	30
```
